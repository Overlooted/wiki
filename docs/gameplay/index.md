# Gameplay Systems

| System | Key Scripts | Summary |
|---|---|---|
| [Items & Appraisal](items-appraisal.md) | `ItemMono`, `AppraisalSystemMono` | Grab items, shake to appraise, tooltip + glow |
| [Economy & Vendors](economy-vendors.md) | `WalletMono`, `VendorMono` | Sell items, earn gold, vendor upgrades |
| [Cart System](cart-system.md) | `CartMono`, `CrateMono`, `WheelControl` | Haul loot, seal crates, cart physics |
| [Tools & Toolbelt](tools-toolbelt.md) | `ToolbeltMono`, `ToolDisplayMono` | Buy, equip, and use tools |
| [Goblin System](goblin.md) | `GoblinMono`, `GoblinSpawner` | Goblins steal loose items from the cart |
| [Hub & Home](hub-home.md) | `PlayerHomeMono`, `HomeItemSlotMono` | Store items between runs |
| [Progression](progression.md) | `ProgressionManager`, `ZonePortalMono` | Zone unlock requirements |
| [Save System](save-system.md) | `SaveManagerMono` | Persist gold, tools, home state |
