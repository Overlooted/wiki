# Pulley Elevator

Load rocks into a counterweight bucket to raise a platform.

## Components

| Component | Role |
|---|---|
| `RockDispenserMono` | Interactable; spawns rocks |
| `PulleyPlatformMono` | Rises as rocks are loaded into bucket |
| `PulleyElevatorMono` | Coordinates platform + bucket physics |

## Setup

1. Place the **Pulley Elevator** prefab
2. Starter rocks are included in the demo scene
3. Adjust rock weight and target platform height on `PulleyElevatorMono`
